js vue node koa 这些技术之间的关系

发表于 2020-08-16 更新于 2021-03-05 分类于 技术的“道”

许多初学者，或者想走向全栈的前端开发者，对 node 和 koa 的关系认识混乱，阻碍了大家技术的进步，所以我打算简述一下这几个技术之间的关系，希望能够对初学者有所帮助。

先声明，此文章仅为了让初学者能够了解 js vue node koa 这些技术之间的关系，对每项技术的深入讨论，并不在本篇文章的写作目的之内，敬请谅解。

js是什么？

JavaScript，简称 JS，是一种高级脚本语言（编程语言），在前端开发中具有绝对的统治地位。

JavaScript 在1995年由 Netscape 公司的 Brendan Eich，在网景浏览器上首次设计实现而成，用来控制浏览器行为，让网页能够实现更复杂的功能。因为 Netscape 公司与 Sun 公司（创造 Java 语言的公司）合作，Netscape 管理层希望它外观看起来像 Java，因此取名为 JavaScript。

1996年8月，微软模仿 JavaScript 开发了一种相近的语言，取名为 JScript（JavaScript 是 Netscape 的注册商标，微软不能用），首先内置于IE 3.0。Netscape 公司面临丧失浏览器脚本语言的主导权的局面。

1996年11月，Netscape 公司决定将 JavaScript 提交给国际标准化组织 ECMA（European Computer Manufacturers Association），希望 JavaScript 能够成为国际标准，以此抵抗微软。ECMA 的39号技术委员会（Technical Committee 39）负责制定和审核这个标准，成员由业内的大公司派出的工程师组成，目前共25个人。该委员会定期开会，所有的邮件讨论和会议记录，都是公开的。

1997年7月，ECMA 组织发布262号标准文件（ECMA-262）的第一版，规定了浏览器脚本语言的标准，并将这种语言称为 ECMAScript。这个版本就是 ECMAScript 1.0 版。

之所以不叫 JavaScript，一方面是由于商标的关系，Java 是 Sun 公司的商标，根据一份授权协议，只有 Netscape 公司可以合法地使用 JavaScript 这个名字，且 JavaScript 已经被 Netscape 公司注册为商标，另一方面也是想体现这门语言的制定者是 ECMA，不是 Netscape，这样有利于保证这门语言的开放性和中立性。

有了 ECMA 组织的加持，JavaScript 很快地统治了众多浏览器，成为了前端开发中最流行的脚本语言。

ECMAscript 5.1 版在2011年6月发布之后，ECMA 就使用了新的命名方式。2015年6月，ECMAScript 6 正式发布，并且更名为“ECMAScript 2015”。之后每年发布一个 ECMAScirpt 的版本，分别为“ECMAScript 2016”，“ECMAScript 2017”等。

但是，很多人还是习惯于使用 ES6 来称呼 ECMAscript 5.1 版之后的版本。所以，如果你听别人说 ES6，其实并非特指“ECMAScript 2015”这一个版本，而是指 ECMAscript 5.1 版之后所有的版本。

总结一下，JavaScript，简称 JS，是前端开发中最流行的脚本语言。JavaScript 的标准由国际标准化组织 ECMA 制定，老版本一般被称为 ES5，具有新特性的版本被称为 ES6。并且如果没有特殊说明，ES6，并非特指“ECMAScript 2015”这一个版本，而是指ECMAscript 5.1 版之后所有的版本。

vue是什么？

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架，全称 Vue.js。通俗的讲，vue 是一个前端框架，使用vue 进行开发，可以大大加快前端的开发速度。

node是什么？

node 全称 Node.js，是一个基于 JavaScript 语法（ES 标准）的服务器端开发语言（类似于JAVA）。Node.js 基于 JavaScript 语言，并且整合了谷歌的 V8 引擎，是一种轻量的、可模块化的开发工具。

Node.js 可以用于 Web 后端服务开发，Web API 服务开发，CLI 工具开发。

koa是什么？

Koa 是一个基于 Node.js 的 web 框架，主要用于开发 web 应用后端服务和 API 服务。

js vue node koa 这些技术之间的关系

Vue 是一个前端框架，里面的脚本部分可以使用 JavaScript 进行开发；Koa 是一个服务器端web框架，用于给 Node.js 在服务器端的开发赋能，加快开发速度；Node.js 是一个服务器端开发语言，开发时的语法基于 JavaScript。

可以总结为，Node.js 给 JavaScript 插上了飞向各端的翅膀，引来了前端圈的大繁荣，进而诞生了 Vue.js 这样的前端框架，进一步加强了前端的繁荣。

从运行时环境解释

JavaScript 是 ES 在浏览器中的一种实现（运行时环境是浏览器），Node.js 是 ES 在服务器中的一种实现（运行时环境是服务器），Koa 是 基于 Node.js 封装的一个框架，Vue 是基于 JavaScript 封装的一个框架
