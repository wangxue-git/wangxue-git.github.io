js vue node koa 这些技术之间的关系

发表于 2020-08-16 分类于 技术的“道”

  许多初学者，或者想走向全栈的前端开发者，对node和koa的关系认识混乱，阻碍了大家技术的进步，所以我打算简述一下这几个技术之间的关系，希望能够对初学者有所帮助。先声明，此文章仅为了让初学者能够了解js vue node koa 这些技术之间的关系，对每项技术的深入讨论，并不在本篇文章的写作目的之内，敬请谅解。
  我先分别介绍一下这几个技术，然后再总述这些技术之间的关系。

js是什么？

  JavaScript，简称JS，是一种高级脚本语言（编程语言），在前端开发中具有绝对的统治地位。
  JavaScript在1995年由Netscape公司的Brendan Eich，在网景浏览器上首次设计实现而成，用来控制浏览器行为，让网页能够实现更复杂的功能。因为Netscape公司与Sun公司（创造Java语言的公司）合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。
  1996年8月，微软模仿JavaScript开发了一种相近的语言，取名为JScript（JavaScript是Netscape的注册商标，微软不能用），首先内置于IE 3.0。Netscape公司面临丧失浏览器脚本语言的主导权的局面。
  1996年11月，Netscape公司决定将JavaScript提交给国际标准化组织ECMA（European Computer Manufacturers Association），希望JavaScript能够成为国际标准，以此抵抗微软。ECMA的39号技术委员会（Technical Committee 39）负责制定和审核这个标准，成员由业内的大公司派出的工程师组成，目前共25个人。该委员会定期开会，所有的邮件讨论和会议记录，都是公开的。
  1997年7月，ECMA组织发布262号标准文件（ECMA-262）的第一版，规定了浏览器脚本语言的标准，并将这种语言称为ECMAScript。这个版本就是ECMAScript 1.0版。
  之所以不叫JavaScript，一方面是由于商标的关系，Java是Sun公司的商标，根据一份授权协议，只有Netscape公司可以合法地使用JavaScript这个名字，且JavaScript已经被Netscape公司注册为商标，另一方面也是想体现这门语言的制定者是ECMA，不是Netscape，这样有利于保证这门语言的开放性和中立性。
  有了ECMA组织的加持，JavaScript很快地统治了众多浏览器，成为了前端开发中最流行的脚本语言。
  ECMAscript 5.1版在2011年6月发布之后，ECMA就使用了新的命名方式。2015年6月，ECMAScript 6正式发布，并且更名为“ECMAScript 2015”。之后每年发布一个ECMAScirpt的版本，分别为“ECMAScript 2016”，“ECMAScript 2017”等。
  但是，很多人还是习惯于使用ES6来称呼ECMAscript 5.1版之后的版本。所以，如果你听别人说ES6，其实并非特指“ECMAScript 2015”这一个版本，而是指ECMAscript 5.1版之后所有的版本。

  总结一下，JavaScript，简称JS，是前端开发中最流行的脚本语言。JavaScript的标准由国际标准化组织ECMA制定，老版本一般被称为ES5，具有新特性的版本被称为ES6。并且如果没有特殊说明，ES6，并非特指“ECMAScript 2015”这一个版本，而是指ECMAscript 5.1版之后所有的版本。

vue是什么？

  Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架，全称Vue.js。通俗的讲，vue是一个前端框架，使用vue进行开发，可以大大加快前端的开发速度。

node是什么？

  node全称Node.js，是一个基于JavaScript语法的服务器端开发语言（类似于JAVA）。Node.js基于JavaScript语言，并且整合了谷歌的V8引擎，是一种轻量的、可模块化的开发工具。Node.js可以用于Web 后端服务开发，Web API 服务开发，CLI 工具开发。

koa是什么？

  Koa 是一个基于Node.js的 web 框架，主要用于开发 web 应用后端服务和 API服务。

js vue node koa 这些技术之间的关系

  Vue是一个前端框架，里面的脚本部分可以使用JavaScript进行开发；Koa是一个服务器端web框架，用于给Node.js在服务器端的开发赋能，加快开发速度；Node.js是一个服务器端开发语言，开发时的语法基于JavaScript。
  可以总结为，Node.js给JavaScript插上了飞向各端的翅膀，引来了前端圈的大繁荣，进而诞生了Vue.js这样的前端框架，进一步加强了前端的繁荣。
